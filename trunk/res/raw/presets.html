Load

Load one of the pre-defined or previously saved fractal generator settings. These settings are lost if the application gets uninstalled. Export all settings you don't want to loose before uninstalling the application.

Save

Name and save the current fractal generator settings. These settings are lost if the application gets uninstalled. Export all settings you don't want to loose before uninstalling the application.

Discard

Discard a previously saved setting.

Import

Import a setting from a file.

Export

Export a setting to a file. Exported settings can be copied to other devices running the Ljapunow Fractals application and imported there to share the best fractals.
