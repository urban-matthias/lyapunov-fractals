ABC Sequence

The periodic sequence of As, Bs and Cs in which the degree of the growth of the population switches, whereas C causes an equation dependent value switch.

Iterations

The number of iterations used to calculate the Ljapunow exponent. The larger the number the better the exponent will be approximated.

Warmup Rounds

The number of initial iterates calculated to get rid of the critical points of the logistic function. Recommended to equal the length of the A-B sequence.

A Interval

The value interval of A for which to draw the fractal. This directly correlates to the x-axis range of the fractal to be drawn.

B Interval

The value interval of B for which to draw the fractal. This directly correlates to the y-axis range of the fractal to be drawn.

X0

The start value of the logistic function.

Equation

Equation to be used for calculating the fractal.

Randomize

Assign most of the parameters with random values.
